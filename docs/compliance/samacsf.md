# SAMA CSF

CHERENKOV supports the **Saudi Arabian Monetary Authority Cyber Security Framework (SAMA CSF)**.

## Mapped Domains

| Domain | CHERENKOV Coverage |
|---|---|
| **Cyber Security Governance** | Full — SHA-256 signed audit traces |
| **Risk Management** | Full — severity-based risk scoring |
| **Vulnerability Management** | Full — automated scanning + TOKAMAK |
| **Cyber Resilience** | Partial — continuous monitoring |
| **Third-Party Security** | Full — wrapper-based assessment |

## Compliance Reports

CHERENKOV generates SAMA CSF-aligned compliance reports with direct control-to-finding mapping, signed with cryptographic trace IDs.
