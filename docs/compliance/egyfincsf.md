# EGY-FIN CSF

CHERENKOV provides native mapping to the **Egyptian Financial Cyber Security Framework (EGY-FIN CSF)**, issued by the Central Bank of Egypt.

## Mapped Domains

| Domain | CHERENKOV Coverage |
|---|---|
| **Vulnerability Management** | Full — automated scanning + TOKAMAK validation |
| **Penetration Testing** | Full — AI-guided attack chains |
| **Security Monitoring** | Partial — trace logging and audit |
| **Incident Response** | Partial — evidence collection |
| **Third-Party Security** | Full — third-party scanner wrappers |

## Audit Evidence

Every EGY-FIN CSF control is mapped to one or more CHERENKOV traces, providing cryptographic proof of compliance.
