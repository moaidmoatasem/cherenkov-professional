# Scanner Strategy

CHERENKOV uses an intelligent scan strategy system that adapts to each target.

## Strategy Types

| Strategy | Description | Use Case |
|---|---|---|
| **Rapid** | Fast, high-signal checks | CI/CD pipelines |
| **Deep** | Comprehensive coverage | Audit / compliance |
| **Stealth** | Low-and-slow, evasion-aware | Red team exercises |
| **Adaptive** | AI-guided, learns from response | Unknown targets |

## How Strategy Works

1. TENSOR analyzes target profile (technology, exposure, compliance reqs)
2. Agent Governor selects optimal strategy
3. KINETIC executes scanners based on strategy priorities
4. Results feed back into LATTICE for future strategy optimization
